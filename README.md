# 渋谷ヒートシム v1.0

渋谷の実地図上で5エリアのWBGT（暑さ指数）を可視化し、すいかばマンが水分補給アドバイスを出すWebアプリ教材です。

## 公開URL
https://shiojima-test.github.io/shibuya-heat-sim/

## 機能
- Leaflet.jsによる渋谷の実地図表示
- 5エリアのWBGT値をリアルタイム色分け表示
- 時間スライダー（6:00〜20:00）で時間変化をシミュレーション
- 再生ボタンで自動アニメーション
- すいかばマンによる暑さレベル別アドバイス
- エリアクリックで詳細ポップアップ（気温・湿度・WBGT）
